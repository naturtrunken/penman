Penetration test manager. Keeps a pentester's notes and thoughts organized.

You can create multiple networks and targets within. For each target, you can add notes for each stage of the pentest cycle.

Hint
  1. Create an account or log in
  2. Create a network.
  3. Use the add_target.sh and add_services.sh scripts to add targets and services directly via the command line.
    • Alternatively, create targets and services via the frontend.
  4. Add notes while you're pentesting.
